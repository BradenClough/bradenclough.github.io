Scatterplots

Scatterplots are a way of visualizing collections of data without a line. Each point on a scatterplot is an individual data entry. In the shown plot, one can see every entry of how a plant's height changes as tempurature increases. This particular plot shows the user how the plants are more prone to being taller as tempurature increases.

Line of Best Fit

The line of best fit is a way of showing the relation between points on a scatter plot. It serves the purpose of estimating how the y value of the graph changes in proportion to the x axis. It is expressed as the formula y = mx+b. The y value is calculated by mulitplying the x value by the slope (m) then adding the y intercept (b). When the code runs, you can see it print the forumla each input's line of best fit, starting with the phrase "The line of best fit is"

Standard Error

A line of best fit's standard error is a way of estimating how far off from the line of best fit each data point will be. It can be found in the code as "The line's standard error is: " It is calculated by averaging the margin of error for all data points. In my code, it appears in a for loop using i as the looping variable. The line of code that calculates it is "SEline += file.height[i] - (m * xColumn[i] + b)"

R Squared

The coefficient of determination, also known as R Squared, is the measure of how much the y axis is impacted by the x axis. It is found by deviding the standard error of the average y value by the standard error of the line of best fit. It can be found in the count printed with the label "R Squared."
